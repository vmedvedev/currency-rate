## Description

BTCUSDT price synchronizer.

A server app which periodically fetch the BTCUSDT price into database and exposes api with exchange rate history and current rate.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
